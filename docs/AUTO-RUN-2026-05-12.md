- uni-app H5 调微信支付不通是预期的(W2 才接小程序壳)
- seed 脚本破坏数据库 → seed 之前 `truncate` 应用相关表,**不** truncate user(保留登录态)
- 任一 Phase 卡 > 半天:**降范围**(例如 Phase I 的 mock-paid 简化为直接 SQL 更新),**不**加班
