# H5 开发指南

> uni-app 编译的 H5 版本,**主要用途**:微信群分享 / 拼猪邀请落地页 / 短信短链。

---

## 一、H5 的定位

**不是主要交互入口**,但**关键场景**:

| 场景 | 为什么需要 H5 |
|---|---|
| 微信群分享拼猪邀请 | 链接打开是 H5,引导下载 APP / 打开小程序 |
| 短信短链 | "你的猪可以出栏了"短信带链接 |
| 抖音 / 公众号 / 朋友圈广告 | 落地页 |
| 客服转化 | 给客户看一个网页比下载 APP 容易 |
| SEO(后续) | 公网可索引,被搜到 |

C 端核心业务流(下单 / 拼猪 / 看猪)**主要靠小程序和 APP**,H5 是流量入口 + 分享物料。

## 五、拼猪分享落地页

### URL 结构

```
https://www.rockingwei.online/h5/?invite=ABC123
```

`?invite=ABC123` 是分享码。

### 页面逻辑

```vue
<!-- src/pages/share/join.vue -->
<template>
  <view class="join-page">
    <view class="hero">
      <text class="title">{{ inviterName }} 邀请你一起认领</text>
      <text class="subtitle">{{ pig.breed }} · {{ pig.farmer_name }}</text>
    </view>

    <LivePlayer :pig-id="pig.id" />

    <view class="info-card">
      <text>主认领人:{{ inviterName }}</text>
      <text>已加入:{{ memberCount }} / 3 家</text>
      <text>分享码:{{ code }}</text>
    </view>

    <!-- #ifdef H5 -->
    <view v-if="!isWechat" class="hint">
      <text>请用微信打开此链接</text>
    </view>
    <button v-else class="join-btn" @tap="join">微信授权并加入</button>
    <!-- #endif -->

    <!-- #ifdef MP-WEIXIN || APP-PLUS -->
    <button class="join-btn" @tap="join">加入拼猪</button>
    <!-- #endif -->
  </view>
</template>

<script setup lang="ts">
import { onMounted, ref } from 'vue';
import { shareApi } from '@/api/share';
import { useUserStore } from '@/store/user';

const code = ref('');
const pig = ref<any>({});
const inviterName = ref('');
const memberCount = ref(0);

onMounted(async () => {
  // #ifdef H5
  const params = new URLSearchParams(location.search);
  code.value = params.get('invite') || '';
  // #endif

  // #ifdef MP-WEIXIN
  const launch = uni.getEnterOptionsSync();
  code.value = launch.query?.invite || '';
  // #endif

  if (!code.value) {
    uni.showToast({ title: '邀请码无效', icon: 'none' });
    return;
  }

  const info = await shareApi.previewByCode(code.value);
  pig.value = info.pig;
  inviterName.value = info.host_name;
  memberCount.value = info.member_count;
});

async function join() {
  const user = useUserStore();
  if (!user.isLogin) await user.wxLogin();
  await shareApi.join(code.value);
  uni.showToast({ title: '加入成功' });
  setTimeout(() => {
    uni.reLaunch({ url: '/pages/pig-detail/index?id=' + pig.value.id });
  }, 1000);
}
</script>
```

## 七、微信 JS-SDK 注入(分享自定义)

如果用户在微信内打开 H5,可以自定义分享卡片:

```ts
// utils/wx-jssdk.ts
// #ifdef H5
import wx from 'weixin-js-sdk';
import { wxSdkApi } from '@/api/wx-sdk';

export async function setupWxShare(opts: {
  title: string;
  desc: string;
  link: string;
  imgUrl: string;
}) {
  if (!isWechatBrowser()) return;

  const config = await wxSdkApi.getSignature(location.href);
  wx.config({
    debug: false,
    appId: config.appId,
    timestamp: config.timestamp,
    nonceStr: config.nonceStr,
    signature: config.signature,
    jsApiList: ['updateAppMessageShareData', 'updateTimelineShareData'],
  });

  wx.ready(() => {
    wx.updateAppMessageShareData(opts);
    wx.updateTimelineShareData(opts);
  });
}
// #endif
```

后端 `/api/wx-sdk/signature` 实现:用小程序 jsapi_ticket 算签名。

## 十、URL 路由

H5 默认 hash 路由 (`/#/pages/home/index`)。建议改成 history 模式(`pages.json`):

```json
{
  "globalStyle": { ... },
  "h5": {
    "router": {
      "mode": "history",
      "base": "/h5/"
    }
  }
}
```

URL 就是:`https://www.rockingwei.online/h5/pages/share/join?invite=ABC123`

nginx 需要 `try_files` 兜底回 `index.html`(history 模式必须)。

## 十二、H5 不做的事

- ❌ 复杂下单流程(转到小程序 / APP 完成)
- ❌ 实时直播(性能差,转小程序)
- ❌ 上传大文件
- ❌ 复杂手势交互

H5 的核心使命是**承接流量 + 引导下载 / 跳转**。
